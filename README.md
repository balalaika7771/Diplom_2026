# Магистерская диссертация: интеллектуальные автономные агенты для диагностики и самовосстановления распределённых систем

Репозиторий содержит исходный код магистерской диссертации в формате LaTeX, оформленной по ГОСТ 7.32–2017, и CI-конвейер для автоматической сборки и публикации PDF.

## Быстрый старт

```bash
# Один раз перед началом работы — инициализация проекта
./setup.sh

# Полная сборка (pdflatex + biber + повторные прогоны)
make

# Быстрая сборка без библиографии
make quick

# Открыть скомпилированный PDF
make view

# Очистить временные файлы
make clean

# Полностью очистить build/ и служебные файлы (включая main.bcf, main.bbl, main.pdf)
make distclean

# Показать ошибки/предупреждения последней сборки
make errors
make warnings
```

Готовый PDF после успешной сборки находится в `build/main.pdf`.

Если при сборке появляется ошибка вида:

```text
ERROR - main.bcf is malformed, last biblatex run probably failed
```

выполните:

```bash
make distclean
make
```

чтобы пересоздать все вспомогательные файлы.

## Структура репозитория

```text
.
├── main.tex           # Основной файл LaTeX (точка входа)
├── config/            # Конфигурация LaTeX (packages.tex, paths.tex, settings.tex)
├── src/
│   ├── title/         # Титульный лист (title.tex)
│   ├── abstract/      # Реферат (abstract.tex)
│   ├── chapters/      # Главы диссертации (01_*.tex … 13_*.tex)
│   ├── executors/     # Сведения об авторе и научном руководителе
│   ├── terms/         # Перечень обозначений и сокращений
│   ├── bibliography/  # База библиографии (references.bib)
│   └── appendix/      # Приложения (appendix*.tex)
├── build/             # Скомпилированный PDF и служебные файлы (генерируются)
├── setup.sh           # Скрипт инициализации проекта
├── Makefile           # Сценарии сборки (см. выше)
└── .github/workflows/ # CI для сборки и релиза PDF
```

Актуальную версию диссертации можно скачать в разделе **Releases** репозитория.
